# StoryRpg Server Setup

A Bash script that simulates setting up a StoryRpg game server through an interactive setup process.

## Features

- Interactive server setup
- Hat, pet, and glasses customization
- Difficulty selection
- World name confirmation
- Maximum player validation (1–15 players)
- Automatic server and player log generation
- Input validation for all user prompts

## Requirements

- Bash
- Linux, macOS, or Windows with WSL

## Run

```bash
chmod +x storyrpg_server_setup.sh
./storyrpg_server_setup.sh
```
